# EnergiConecta — Mock (React + Tailwind)
Versión mock funcional de la plataforma EnergiConecta.

## Ejecutar localmente
1. `npm install`
2. `npm run dev`
3. Abrir la URL que indique Vite (ej. http://localhost:5173)

Las imágenes son SVG placeholders incluidos en /public. Reemplázalas por fotos reales si lo deseas.
